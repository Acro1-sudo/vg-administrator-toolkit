  • To begin, type a name in the search bar and press Enter or click the 'Search' button. Searches are not case-sensitive.
  • The left table shows admin command logs. The right table shows all playernames with their first login date (starting May 1st 2020, PR 1.6 release).
  • The left table shows all logs containing your input string, in chronological order. This means you can search your own name too,
    in case you want to see your recent actions.
  • The right-hand table will show all usernames that contain your search string, together with their alternate usernames.
    Usernames grouped together between grey lines are on the same hashkey. (Search for 'papa' to see a clear example.)
  • Click any name to show you that name's logs. Only the clicked player and accounts on that same hashkey will be shown
    until you do a new search.
  • You can leave the username field blank and search by command (warn, kick,...) only, but this is heavy on RAM usage!
    It will render at least 30.000 table rows. If you have a potato computer, this is not advisable.
    For this reason, searching with empty name field and empty filter will simply clear the page.
